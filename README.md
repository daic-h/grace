# Grace

This is a Theme for [Lokka](http://lokka.org)

Is required [lokka-tag_cloud](https://github.com/daic-h/lokka-tag_cloud).

## Install

    $ cd public/theme
    $ git clone git://github.com/daic-h/grace.git
    $ git clone git://github.com/daic-h/lokka-tag_cloud.git
